📊 第一章 · 期权交易策略

1.1 期权基础概念

期权是一种金融衍生品，赋予持有者在特定日期或之前以特定价格买卖标的资产的权利（而非义务）。

📈 看涨期权 (Call)

你有权在到期日或之前以行权价买入标的资产。当你预期股价上涨时买入。

Long Call

📉 看跌期权 (Put)

你有权在到期日或之前以行权价卖出标的资产。当你预期股价下跌时买入。

Long Put

💰 期权费 (Premium)

购买期权时支付的价格，由内在价值 + 时间价值组成。这是你的最大损失（对买方而言）。

⏰ 时间衰减 (Theta)

期权每天会损失一部分时间价值，越临近到期衰减越快。这是期权买方的死敌，卖方的朋友。

📐 关键公式
内在价值 (Call) = Max(0, 标的价格 - 行权价)
内在价值 (Put) = Max(0, 行权价 - 标的价格)
时间价值 = 期权价格 - 内在价值

1.2 四大基础策略

策略操作市场看法最大盈利最大亏损
买入 Call支付权利金买 Call看涨理论上无限全部权利金
买入 Put支付权利金买 Put看跌行权价 - 标的价格 - 权利金全部权利金
卖出 Call收取权利金卖 Call中性/看跌全部权利金理论上无限 ⚠️
卖出 Put收取权利金卖 Put中性/看涨全部权利金行权价 - 权利金
⚠️ 风险提示

卖出裸期权（Naked Option）风险极大，尤其是 Naked Call：如果股价暴涨，你的亏损理论上是无限的。新手备兑开仓（Covered Call）至少要有对应的正股持仓。

1.3 进阶组合策略

🔀 牛市价差 (Bull Call Spread)

买入低价 Call + 卖出高价 Call（同到期日）。降低成本，但封顶收益。适合温和看涨。

温和看涨

🔁 熊市价差 (Bear Put Spread)

买入高价 Put + 卖出低价 Put。成本有限，收益有限。适合温和看跌。

温和看跌

🦅 铁鹰价差 (Iron Condor)

卖出一组价差 + 买入更远价差。四个腿。预期股价横盘，收取权利金。风险有限。

区间震荡高级策略

📅 日历价差 (Calendar Spread)

卖出近期 Call + 买入远期 Call（同行权价），赚取时间衰减差异。

时间价值

🔗 跨式组合 (Straddle)

同时买入同行权价的 Call 和 Put。预期大波动但方向不明。成本 = 两个权利金之和。

波动率交易

📐 带式/条式 (Strip/Strap)

Straddle 的变体：Strip 多一个 Put（偏空），Strap 两个 Call（偏多）。

方向性+波动

1.4 希腊字母（Greeks）

Δ Delta

标的价格变动 1 元，期权价格变动多少。Call Δ ∈ (0,1)，Put Δ ∈ (-1,0)。实值越深 |Δ| 越接近 1。

Γ Gamma

Delta 的变化率。衡量 Delta 的稳定性。平值附近 Gamma 最大，临到期时急剧上升。

Θ Theta

时间衰减。每天期权价格因时间流逝损失的金额。卖方为正（收时间钱），买方为负。

ν Vega

隐含波动率上升 1%，期权价格变动多少。波动率交易的核心指标。临到期时 Vega 最小。

  2. 首次申请期权权限从 Level 1（买 Call/Put）开始，不要一上来就做裸卖空
  3. 理解你策略的"最坏情况"再下单。画好盈亏图。
  4. 不要持仓到期！提前 1-2 周关闭或展期

📉 第二章 · 技术分析精要

2.1 趋势与形态

技术分析三大假设：① 市场行为涵盖一切信息 ② 价格沿趋势移动 ③ 历史会重演。

📈 上升趋势

高点和低点逐步抬高。回调时买入。在上升趋势中，"逢低做多"是最安全的方式。

📉 下降趋势

高点和低点逐步降低。反弹时做空或观望。下跌趋势不要接飞刀。

↔️ 区间震荡

价格在支撑和阻力间反复。低吸高抛。突破追随。

📊 经典价格形态（建议在 thinkorswim 图表功能中查看）
形态类型含义
头肩顶反转（顶部）上升趋势终结信号。跌破颈线确认目标价。
头肩底反转（底部）下降趋势终结信号。突破颈线确认。
双顶 M反转两次上攻失败，看跌。量能递减更佳。
双底 W反转两次探底成功，看涨。
上升三角形持续水平阻力 + 上升支撑，通常向上突破。
下降三角形持续水平支撑 + 下降阻力，通常向下突破。
旗形/三角旗持续趋势中继形态，突破后量度移动。

2.2 核心技术指标

📊 移动平均线 (MA)

SMA： 简单移动平均 / EMA： 指数移动平均（更看重近期）

常用组合：MA20 / MA50 / MA200

金叉：短线上穿长线 → 买入信号

📏 MACD

指数平滑异同平均线。DIF - DEA = MACD 柱

信号：① 金叉/死叉 ② 背离 ③ 零轴穿越

⚡ RSI (相对强弱)

0-100 区间。>70 超买，<30 超卖

背离信号：价格新高 + RSI 新低 → 看跌背离

📐 布林带 (Bollinger)

中轨=MA20，上下轨=MA20 ± 2σ

触及上轨→可能超买，触及下轨→可能超卖

带宽收窄 → 大波动即将来临

📊 成交量 (Volume)

量在价先。突破放量 = 有效突破

无量上涨 = 可能诱多

📈 VWAP

成交量加权平均价。机构交易基准

价格 > VWAP = 多头优势。日内交易核心指标。

2.3 K 线形态心理学

K 线不仅仅是价格记录，它记录了当天多空双方博弈的过程。理解形态背后的市场心理比死记硬背更重要。

🔨 锤子线

下影线长，实体小。抛压被大量接盘歼灭，可能见底。需次日阳线确认。

⭐ 十字星

开盘=收盘。多空犹豫不决。顶部十字星是变盘信号，"射击之星"看跌。

📦 吞没形态

大实体完全覆盖前一根。看跌吞没 → 顶部反转，看涨吞没 → 底部反转。

👥 三白兵/三乌鸦

连续三根同色实体。三白兵看涨，三乌鸦看跌。反映市场情绪一边倒。

💡 技术分析的正确使用方式

不要孤立使用单一指标！至少需要 2-3 个指标共振确认。例如：价格突破 MA20 + 放量 + RSI 从超卖回升 → 三重确认的做多信号。

🛡️ 第三章 · 资金管理与风控

3.1 凯利公式 (Kelly Criterion)

凯利公式告诉你：在一笔有优势的赌局中，应该下注多少比例才能使长期收益最大化。

📐 凯利公式
f* = (bp - q) / b
其中：
　f* = 最佳下注比例
　b = 赔率（盈利/本金）
　p = 胜率
　q = 1-p（败率）

例：胜率 60%，盈亏比 2:1
　f* = (2×0.6 - 0.4) / 2 = 0.4 → 投入 40%
💡 实际操作建议

使用"半凯利"（Half Kelly） 更安全——把凯利公式算出来的数字减半。比如凯利值 40%，实际下注 20%。这样即使胜率估计偏差，也有缓冲空间。

🔺 金字塔加仓

盈利时逐步加仓，但每次加仓金额递减。比如首次 5%，第二次 3%，第三次 1%。这样可以拉高均价但不至于过度集中。

🔻 倒金字塔减仓

高位先卖大份额，跌了卖少一点。大涨时锁定大部分利润，留一点博更多。

🎯 固定比例法

每次交易固定投入总资金的一定比例（如 5-10%）。简单机械，适合严格执行。

⚖️ 风险平价

根据每只股票/资产的历史波动率调整仓位，让每笔交易的风险贡献相等。

3.3 尾部风险管理

尾部风险是指那些极不可能发生但一旦发生影响巨大的事件（如 2008 年金融危机、2020 年疫情暴跌）。

🛡️ 保护性看跌 (Protective Put)

持有正股的同时买入 Put，相当于为股票买保险。成本 = 权利金。

保险策略

📊 VIX 恐慌指数

流动性优先

地理分散
⚠️ 风险管理铁律
  1. 永远不要 All In： 任何单一头寸都不应该让你夜不能寐
  2. 止损是纪律： 预设止损点并严格执行，不允许"再等等"
  3. 活下来第一： 巴菲特说："投资第一条规则是不要亏钱，第二条规则是记住第一条。"
  4. 波动率 ≠ 风险： 真正的风险是永久性本金损失，而非账面浮亏

4.2 再平衡方法

📅 时间再平衡

每季度或每年检查一次，卖出涨多的，买入跌多的，恢复原始比例。

🎯 阈值再平衡

当某类资产偏离目标配置超过阈值（如 ±5%）时才触发再平衡。减少交易次数和费用。

常用

💡 再平衡溢价

📊 Fama-French 三因子 → 五因子模型

Eugene Fama（诺贝尔奖得主）的研究发现：股票收益可以用市场风险、规模（小盘 vs 大盘）和价值（低估值 vs 高估值）三个因子来解释。后来加入了盈利能力和投资水平，扩展为五因子模型。

🧠 第五章 · 行为金融陷阱

5.1 认知偏差

🔍 确认偏差 (Confirmation)

只寻找支持自己观点的信息，忽略反面证据。关注好消息，无视坏消息。

最常见

😱 损失厌恶 (Loss Aversion)

亏损 $1000 的痛苦 ≈ 盈利 $2000 的快乐。导致我们死扛亏损仓、过早卖出盈利仓。

🐑 羊群效应 (Herding)

别人买我买，别人卖我卖。FOMO（害怕错过）驱动追高杀跌。

📍 锚定效应 (Anchoring)

被买入价格"套牢"，决策围绕成本价而非基本面变化。卖掉赚钱的、死扛亏钱的。

📈 近因偏差 (Recency)

过度外推最近的趋势。牛市时认为永远涨，熊市时认为永远跌。

🎲 赌徒谬误 (Gambler's)

连续 5 次红色后"应该出黑色"了——每次独立概率 50%，历史不影响未来。

5.2 交易心理建设

📋 交易计划

每次交易前必须写清楚：① 为什么买 ② 目标价 ③ 止损价 ④ 仓位。盘中按计划执行，不允许临时决策。

📓 交易日志

记录每笔交易：入场理由、情绪状态、执行质量、结果复盘。每周回看一次。

🧘 冥想与情绪觉察

每天用 10 分钟观察自己的情绪。发现 FOMO、恐惧、贪婪时暂停交易，等冷静再回来。

📊 概率思维

接受每一笔交易都可能错。你不是在"预测"，你是在有正期望值的概率下注。长期坚持就够了。

⚠️ 投资中最难的事

不是分析公司、不是选策略，而是控制自己。市场先生的出价只能利用，不能预测。长期成功的投资者都有一个共同点：纪律。

6.1 策略开发流程

发现 α 来源

α = 超额收益。可以来自：因子（动量/价值）、技术信号、基本面、另类数据（如情感分析）。

构建信号

将想法转化为可计算的数学公式/条件。例如：MA5 上穿 MA20 + RSI < 30 → 买入信号。

回测

用历史数据验证信号的有效性。关注：年化收益、最大回撤、夏普比率、胜率。

参数优化与稳健性

避免过拟合。策略在不同时段、不同市场、不同参数下都应该有效。样本外测试。

执行与监控

模拟交易 → 小资金实盘 → 逐步放量。持续监控策略衰减。
